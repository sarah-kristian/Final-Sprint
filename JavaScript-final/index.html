JSON, JavaScript, and MadLibs


Overview

This project is all about using JSON data to create strings. To this end, I have developed madLibs-style story generator. We'll use a JSON file to get word lists for nouns, adjectives, and verbs, and then use those lists to create a story. We'll also create a class to help us manipulate the story.

Steps 1 - 4: Getting Data from JSON file

Here, I'm using fetch to get data from a JSON file and display it on the page. I'm also using a for loop to display the data in the console. To see this, you can open the developer tools on your browser.

Step 5: Functions for displaying JSON content

The JSON file contains word lists for nouns, adjectives, and verbs and default values for a story. We can use these to along with a class to play something like madLibs!

To do this, we create storyVersion class that includes a function to display data from the JSON file in a template literal. We'll see how plays out in a moment, but first let's create two instances of the story: the defaultVersion and the randomVersion.

The defaultVersion takes preset default values from the JSON file.
